# Stanok_DeliveryClub


Описание решаемых задач данного проекта:
Создается заказ, после чего требуется чтобы в течении N минут статус заказа был изменен, иначе заказ отменяется. Сделан отдельный сервис, который для каждого заказа создает таймер на N минут, учитывая возможное падение и перезапуск приложения (восстанавливает все незавершенные таймеры). Если статус был изменен, таймер диспоузится. 

Написаны функциональные тесты для работы таймеров (автоматически создается контейнер с тестовым PostgreSQL бд, тестирование проводится изолированно от основного бд).
